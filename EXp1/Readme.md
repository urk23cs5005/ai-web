# Event Registration & Promotional Portal
This repository contains the front-end code for the Tech-Innovate 2025 event website. It features event highlights with various media types and a comprehensive registration form. 
Features:
* Promotional Media: Includes sections for an event banner image, a promotional video (vid1.mov), and an introductory audio clip (audio.mp3).
* Registration Form: A detailed form for attendees to register, capturing their full name, email, preferred session date, attendee type (Student/Professional), workshop choice, motivation, and an option to upload an ID proof.
* Responsive Design: (Assumed via viewport meta tag) Basic structure for a modern web portal.
* Validation: Basic HTML5 form validation for required fields like name, email, and date.
